In order to use the Demo1.py you need to make sure to have access to the calibration matrices located in the Calibration folder on the Pi
All the code requires then is a camera plugged in. Run it through idle. The camera will then have a constant background video feed.
On that video feed there will be a constant aruco marked detection ran. If it detects one it will then calculate the angle and display it in the terminal
or on the LCD screen.
